Template Structure

To get to know what you've bought.

Eight template
  • index.html
    • /css/ (all the styles)
    • /fonts/ (font icons)
    • /images/ (your images)
    • /js/
      • functions.js (template functions)
      • /assets/ (libraries/assets that are needed)
    • /modules/ (modules like the Tera Lightbox)
    • /pages/ (the pages included with Eight)
    • /php/ (mail script)

HTML Structure

Most pages have a structure like this:

<body> // Add your favorite helper classes
  <div class="container-fluid"> 
    <header> 
      // Logo, Menu and Search
    </header> 
    <section class="hero"> 
      // Hero
    </section> 
    <section class="grid-container"> 
      // The portfolio or blog grid
    </section> 
    <footer> 
      // The footer
    </footer> 
    <footer class="bottom"> 
      // The copyright footer (on the very bottom)
    </footer> 
    // Scripts & generated vertical lines
  </div> 
</body>

CSS Structure

The Elements.css file has all the styled elements used in Eight. You can modify this stylesheet at your own risk.

To adjust styles, it's better to override them using the custom.css file, also included with Eight.
