Filter options

How it works

It's easy as adding a link inside the filter container, using a "data-filter" attribute that corresponds with the class that needs to be filtered. You can then use this class on every ".item"

Markup
// The Filter container
<div class="filter-container">
  <div class="container">
    <div class="inner">
      <ul class="filters"> // The filter list
        <li>
          // You can select all items by simply filtering on ".item"
          <a class='filter' href='#' data-filter='item'>All</a>
        </li>
        <li>
          // This filter corresponds with items that have a ".design" class
          <a class='filter' href='#' data-filter='design'>Design</a>
        </li>
